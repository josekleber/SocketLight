Welcome to Socket Light project.

This project is study for the construction of classes for communication via socket in C++11.

It will initially only supported the TCP over IPv4, but it is my purpose to leave this class ready to work with UDP and accept the IPV6.

One of my studies, along with the socket, is the use of semaphores and the possibility of working with concurrent calls to the socket.

Feel free to contribute if they wish and use of course ...
